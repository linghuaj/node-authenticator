# Authenticator Starter

Basic Express authentication using ES7, Passport, and MongoDB.

### Env Setup:
1.  [BabelJS] (https://gist.github.com/vanessachem/d071027fad39507b9c9f)
2.  MongoDB


### To run: 
```
npm start
```
